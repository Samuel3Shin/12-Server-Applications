# 간단한 페이스북

사용자 인증 메커니즘과 타임라인 기능,
친구를 추가하고 친구의 타임라인을 출력하는 기능 구현


실행 명령어 <br>
$ npm install <br>
$ npm run build       // 웹팩 모듈을 이용해서 /src/index.js 를 /public/bundle.js 로 변환한다! <br>
$ npm start <br>
